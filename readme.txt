Данный проект представляет собой Telegram-бота для рассылки уведомлений о новых
заказах на биржах фриланса.

Текущая версия реализует поддержку сайтов следующих бирж:
    - https://www.fl.ru;
    - https://freelance.ua.

Возможности бота:
    - отправка сообщений как в чат Telegram, так и на e-mail;
    - фильтрация уведомлений либо по категориям, либо по ключевым словам;
    - точное отображение структуры категорий проектов, как на соответствующих
      биржах фриланса (меню бота формируется динамически);
    - настройка бота осуществляется через интерфейс inline-меню Telegram.

Ограничения: данная версия бота не предназначена для масштабного развёртывания.
Предположительно, допустимая нагрузка — не более нескольких десятков активных
пользователей (в зависимости от интервала рассылки уведомлений).

Для запуска/перезапуска и вызова меню бота предназначена команда /start
(других команд в текущей версии нет).

Использован стек технологий:
    - python-3.8.6;
    - sqlite3-3.32.3;
    - requests-2.24.0;
    - beautifulsoup4-4.9.3;
    - aiogram-2.10.1.
